# CompEngiBot

**CompEngiBot** — це Telegram-бот, який дозволяє отримувати випадкові мотиваційні цитати та додавати власні. Бот зручний для швидкого надихання та збереження улюблених цитат.

---

## Встановлення

1. Клонувати репозиторій:

```bash
git clone <посилання_на_репозиторій>
cd <назва_папки>
```

2. Встановити необхідні геми через Bundler:

```bash
bundle install
```

3. Запустити бота:

```bash
ruby bot.rb
```

> Перед запуском переконайтеся, що у файлі `bot.rb` вказано ваш **TOKEN** від Telegram Bot API.

---

## Використання

Бот підтримує такі команди:

| Команда      | Опис |
|-------------|----------------------------------------------------|
| `/start`    | Запустити бота та активувати сесію.             |
| `/quote`    | Отримати випадкову цитату з бази.               |
| `/addquote` | Додати власну цитату (поетапно: текст → автор). |
| `/stop`     | Припинити сесію з ботом.                        |

При додаванні цитати бот перевіряє коректність введеного тексту та не дозволяє додавати порожні або «сміттєві» повідомлення.

---

## Посилання на бота

> Бот працює локально. Щоб запустити його, необхідно мати власний Telegram-бот і токен.
